# 513 Sips Website

A stunning, highly animated website for 513 Sips mobile bartending service.

## 🚀 Live Site
https://egady513.github.io/513sips-tools/website.html

## 📁 Folder Structure

```
projects/513sips/
├── website.html          # Main website file
├── README.md             # This file
├── images/
│   ├── hero/            # Hero background images/videos
│   │   └── (add your hero images here)
│   └── gallery/         # Gallery images
│       └── (add your gallery images here)
```

## 🖼️ Adding Photos

### Hero Background
1. Add your hero image or video to `/images/hero/`
2. Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`, `.mp4`, `.webm`
3. The first file found will be used automatically
4. For best results, use high-resolution images (1920x1080 or larger)

### Gallery Images
1. Simply drop any images into `/images/gallery/`
2. Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`
3. Images appear automatically - no code changes needed!
4. The gallery reads ALL files from this folder

## ✨ Features

### Animations
- Scroll-triggered reveal animations (GSAP)
- Parallax effects on hero section
- Hover micro-interactions on all buttons and cards
- Floating background elements
- Smooth page transitions

### Gallery
- Automatic image loading from folder
- Lightbox with keyboard navigation (arrow keys, ESC)
- Zoom effects on hover
- Share buttons on gallery items
- Supports both images and videos

### Social Media
- Instagram feed embed section
- Social links in footer
- Share functionality

### Contact Form
- Fully functional inquiry form
- Responsive design
- Event type selection

## 📱 Responsive Design

The website is fully responsive and optimized for:
- Desktop (1024px+)
- Tablet (768px - 1024px)
- Mobile (< 768px)

## 🛠️ Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, Grid, Flexbox
- **JavaScript** - Vanilla JS
- **GSAP** - Advanced animations and scroll triggers
- **No build step required** - Pure static files

## 📝 Customization

### Update Contact Information
Edit the contact section in `website.html`:
- Email address
- Phone number
- Location
- Social media links

### Update Instagram Handle
Change `@513sips` in the Instagram section to your actual handle.

### Add/Remove Services
Edit the services grid in the Services section.

## 🚀 Deployment

This site is configured for GitHub Pages:
1. Push changes to the `main` branch
2. GitHub Pages will automatically deploy
3. Site will be live at your configured URL
